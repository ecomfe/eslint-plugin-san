# san/one-component-per-file

要求每个文件中只有一个组件

  • ⚙️ 此规则包含于 "plugin:san/strongly-recommended" 和 "plugin:san/recommended".

# 📖 规则细节

此规则会检查每个文件是否只有一个组件。

/* ✗ BAD */ export class UIAlert extends san.Component { // ... } export class UIAlert extends san.Component { // ... }
Now loading...
<script> /* ✓ GOOD */ export default { name: 'ui-alert' } </script>
